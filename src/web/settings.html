    Advanced — E2EE compliance mode

    Gravio scans are encrypted client-side before publishing. You can configure the encryption mode per-scan via CLI flags for compliance workflows (e.g. finance, health, government). Use the decrypt tool below to manually recover plaintext runs in your dashboard for audit or recovery.

    Configure E2EE mode on scan:

    • node gravio.mjs --token gv_xxx — Default (API-key derived). Auto-configured on first run.
    • node gravio.mjs --passphrase "my-secret" — Passphrase mode. You control the encryption key.
    • node gravio.mjs --key "<64-hex>" — Raw key mode. For custom key management.

    All modes encrypt with AES-256-GCM. Server stores ciphertext only.
