Download the Gravio Scanner

One JavaScript file. Drop it in any folder. Requires Node.js 18+ — that's all.

gravio.mjs Latest

Download gravio.mjs
macOS / Linux
curl -fsSL https://gravio.dev/cli/gravio.mjs -o gravio.mjs
Windows (PowerShell)
Invoke-WebRequest https://gravio.dev/cli/gravio.mjs -OutFile gravio.mjs

Check your Node version first: node --version — should be v18 or higher.

Quick start

  1. Download — run one of the commands above in any project folder.
  2. Scan once — score your agent project right now: 
    node gravio.mjs --once --target .
    Results are written to agent-quality/runs/latest.json.
  3. Publish (optional) — send encrypted results to your Gravio dashboard: 
    node gravio.mjs --once --target . --publish \
  --project my-project \
  --server https://gravio.dev \
  --api-key gv_your_key_here
  4. Watch mode — re-scan on every file change: 
    node gravio.mjs --target .
    Press Ctrl+C to stop.

What does it scan?

The scanner inspects your project's file structure and git metadata — it never reads secret values. It produces a structured run JSON that the Scoring Tool uses to evaluate your agent against five quality dimensions:

  • Safety — prompt injection guards, secret hygiene
  • Reliability — retry logic, error handling, circuit breakers
  • Evaluation — presence of evals, golden datasets, regression tests
  • Observability — traces, spans, structured logs
  • Governance — model pinning, version metadata, audit trails

Privacy & access control

The scanner only reads file structure and git metadata — it never reads the content of your source files, environment variables, or secrets. Your raw code stays on your machine.

When you publish results, access is controlled by your account and API key. Only authenticated sessions can retrieve your project's scorecard data.

Need stronger guarantees? End-to-end encryption (E2EE) is available as an optional feature on Pro and Team plans — your key is generated locally and the server stores only ciphertext it cannot read.

The scanner itself is open source. Inspect gravio.mjs — it's a readable JavaScript bundle.
