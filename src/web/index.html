Agent Scorecard Platform

Evaluate agent run artifacts with guided JSON templates, validation, and a no-code builder

Quick Start

  1. Paste a run artifact in Run JSON, or click Load realistic sample.
  2. Optionally paste a previous run artifact to enable regression gate checks.
  3. Click Evaluate to compute score, gate outcomes, and dimension breakdown.

Required shape for Run JSON

{
  "runId": "string",
  "scorecard": {
    "safety": 0-100,
    "reliability": 0-100,
    "evaluation": 0-100,
    "observability": 0-100,
    "governance": 0-100
  },
  "workflowResults": [{ "id": "x", "status": "pass|fail" }],
  "adversarialResults": [{ "id": "llm01", "status": "pass|fail" }]
}
