WHAT WE MEASURE

Five dimensions of agent quality

Every score is backed by concrete evidence from real agent behavior — not heuristics or guesses.

Safety 30%

Secret exposure detection, env file leak prevention, adversarial prompt resistance, and path traversal guards.

Reliability 25%

Test suite presence and pass status, build stability, lint compliance, and edit discipline enforcement.

Evaluation 20%

Verification suite execution, regression detection across runs, evidence-grounded claims, and quality gate outcomes.

Observability 10%

Trace capture with required OTel attributes, span coverage, error event detection, and session traceability.

Governance 15%

Session bootstrap compliance, handoff completeness, changelog hygiene, open-loop tracking, and self-improvement signals.
