GETTING STARTED

From zero to first score in 5 minutes

Download the CLI, authorize your folder once with your user-bound API key, then run a local scan. The first scan creates your project automatically — no naming step needed.

Steps 2 & 3 include exact CLI commands. Create a free account or sign in once to unlock copy-ready commands with your API key pre-filled.

Step 1

Download the Gravio CLI

One file, no install. Drops gravio.mjs into your current folder. Then run setup once to verify Node.js 20+ and install detected repo dependencies.

macOS / Linux:

curl -fsSL https://gravio.dev/cli/gravio.mjs -o gravio.mjs

Windows (PowerShell):

Invoke-WebRequest https://gravio.dev/cli/gravio.mjs -OutFile gravio.mjs

Run setup once (all OS):

node gravio.mjs --setup --target .

Step 2

Authorize this folder once

Your user-bound API key is auto-filled below. Pick a project name — the project will be created automatically on your first scan.

node gravio.mjs --authorize --target . --project my-project --server https://gravio.dev --api-key gv_sign_in_to_auto_fill_token
Sign in to reveal command with your API key

Authorization is saved locally in .gravio/auth.json. No need to re-authorize on subsequent scans.

Step 3

Run your local scan

From the same folder where you downloaded gravio.mjs, run:

node gravio.mjs --once --target .
Sign in to reveal command

First scan: automatically detects it's a new project and creates it. Subsequent scans: detects the existing project and adds results to it. Scans are encrypted and auto-published securely — no local JSON artifact is written.

Common questions

Can I use Gravio without coding skills?

Yes. Use the guided steps above. If commands are new to you, copy one command at a time and paste into terminal exactly as shown.

Do I need to install anything?

No. gravio.mjs is a single self-contained file with zero npm dependencies. The only requirement is Node.js 20 or newer (download here). Drop the file in any folder and run it.

What does the local scan actually look at?

It walks the target folder for evidence of secret hygiene, test signal, changelog, session notes, and git status — then encrypts and publishes to cloud. It never reads .env contents and requires folder authorization first.

How does decryption work on the dashboard?

Free tier shows generic rating only. Upgrade to Pro or Team to unlock detailed remediation guidance and full decryptable scorecard details.

Why was I getting logged out?

Switching between domains can break session continuity. We now keep onboarding and auth on gravio.dev to keep sessions stable.
