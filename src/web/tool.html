Agent Scorecard BETA

Score an AI-agent run against safety, reliability, evaluation, observability, and governance gates. No data leaves your browser — evaluation runs locally.

How to get a run JSON

Three ways, easiest first. Pick one and the textarea below will fill in.

1

Try a sample (fastest)

Click Load realistic sample below to drop in a complete, valid run JSON you can score immediately. Use Load minimal sample for the bare-minimum shape.

2

Use the Quick builder

Scroll down to the Quick builder card. Enter scores and pass/fail counts — it generates a valid run JSON for you. Best when you already know your numbers but don't want to write JSON by hand.

3

Generate one from your own project

Run the Gravio scanner against your repository — it writes a fresh latest.json you can paste here.

node scripts/scanner-daemon.mjs --once

The result lands at agent-quality/runs/latest.json. Open it, copy the contents, paste into the textarea below — or use Import JSON file.

Required JSON shape (advanced) 
{ runId, scorecard: { safety, reliability, evaluation, observability, governance },
  workflowResults: [{ id, status }], adversarialResults: [{ id, status }] }
